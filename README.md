## Usage

prep work:
```bash
~ bundle install
~ chmod +x ./shopping_cart
```

### Running test suite

`~ ./shopping_cart test`

### Running in interface mode

`~ ./shopping_cart`

### Running in IRB mode

`~ ./shopping_cart irb`
